# Security policy

## Supported version

| Version | Supported |
|---|---|
| 5.4.x | Yes |
| Older versions | No |

## Reporting a vulnerability

Do not open a public issue containing credentials, exploit details or private
data. Report the vulnerability privately through GitHub Security Advisories:

`https://github.com/SmokerGreenOG/ToolCase/security/advisories/new`

Include the affected tool, reproduction steps, impact and a suggested fix when
available.
